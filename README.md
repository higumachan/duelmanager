# duelmanager

## Set firebase information

`config/firebase-config.js`に対してFirebaseの設定を行ってください。

```
{
        API_KEY: '...',
        AUTH_DOMAIN: '...',
        DATABASE_URL: '...',
        PROJECT_ID: '...',
        STORAGE_BUCKET: '...',
        MESSAGING_SENDERID: '...',
        APP_ID: '...',
        MESUREMENT_Id: '...' 
    }
```
